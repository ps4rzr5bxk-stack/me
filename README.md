# me-bot

Telegram бот с обязательной подпиской на каналы.

## Переменные окружения
- `BOT_TOKEN` — токен бота от @BotFather
- `ADMIN_ID` — твой Telegram ID (узнать у @userinfobot)

## Команды админа
- /admin — панель управления
- /add_channel @username Название — добавить канал
- /remove_channel @username — удалить канал
- /list_channels — список каналов

## Важно
Добавь бота как администратора в каждый канал!
